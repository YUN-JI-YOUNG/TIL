## 메모리   

- 10진수 : 0 ~ 9    
- 2진수(1bit) : 0 ~ 1     
- 16진수(2^4, 4bit = 1byte) : 0 ~ 9 + A ~ F  

### 1. 16진수        
##### 16진수 사용처 -> 컴퓨터 메모리, RGB 값 표시(R 2bit, G 2bit, B 2bit)      
- 16진수 사용시, 다른 진수와 헷갈리지 않도록 숫자 앞에 0x 붙이기로 약속. 0x0, 0x1 ...0x1A...
